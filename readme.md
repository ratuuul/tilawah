# Tilawah — Random Quran Ayah

A minimal, beautiful **Qur'an random ayah generator** built as a **fully static web app**.  
It displays Qur'an verses with **Arabic text, translation, tafsir, and navigation controls** in a clean UI.

🌐 Live site:  
https://ratuuul.github.io/tilawah

---

## ✨ Features

- 🎲 **Random Ayah Generator**
- 📖 **Arabic text with Tajweed color rendering**
- 🌍 **English translation**
- 📚 **Tafsir support (toggle on/off)**
- ⬅️➡️ **Previous / Next Ayah navigation**
- 🔖 **Bookmark system (saved locally)**
- 🌙 **Beautiful dark UI**
- ⚡ **Fast static site (no backend required)**

---

## 🧩 Tech Stack

- **HTML**
- **CSS**
- **Vanilla JavaScript**
- **Qur'an API for ayah data**

No frameworks. No build tools. Just static files.
